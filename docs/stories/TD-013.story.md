## Description

Configurar Storybook para documentação visual e desenvolvimento isolado de componentes do Health Dashboard.

### Problem Statement

Não havia ambiente de documentação visual para componentes, dificultando o desenvolvimento isolado, testes visuais e documentação para outros desenvolvedores.

## Dev Notes

### Dependências Necessárias

```bash
npm install -D @storybook/react-vite @storybook/addon-links @storybook/addon-essentials \
  @storybook/addon-interactions @storybook/addon-a11y @storybook/addon-themes
```

### Estrutura

```
.storybook/
├── main.js      # Addons, framework config
├── preview.jsx  # Global decorators, parameters
└── manager.jsx  # UI theme, sidebar config

src/stories/
├── Card.stories.jsx
├── StatusBadge.stories.jsx
├── HealthScore.stories.jsx
├── LoadingSpinner.stories.jsx
└── ThemeToggle.stories.jsx
```

### Comandos

```bash
npm run storybook        # Desenvolvimento (porta 6006)
npm run build-storybook  # Build estático
```
