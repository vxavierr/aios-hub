## Core Concepts

### DNA (Personality DNA)

A structured, portable representation of a personality that captures:
- Linguistic patterns (vocabulary, sentence structure, phrases)
- Emotional characteristics (sentiment, tone, triggers)
- Cognitive style (reasoning, decision-making)
- Knowledge domains (expertise areas, terminology)
- Communication patterns (formality, humor, engagement)

**Properties:**
- Versioned (schema version for migrations)
- Portable (JSON/YAML format, <500KB)
- Fidelity-scored (0-100 completeness metric)

### Manifest

A deployment-ready configuration that translates a DNA profile into:
- Provider-specific system prompts
- Model parameters (temperature, max_tokens)
- Context window configuration
- Response formatting rules

### Pipeline Stages

| Stage | Input | Output | Description |
|-------|-------|--------|-------------|
| **Extract** | Raw sources (YouTube, PDF, Web) | Normalized content | Data ingestion and normalization |
| **Analyze** | Extracted content | Trait analysis | Pattern detection and analysis |
| **Synthesize** | Analysis results | DNA profile | Consolidation into DNA |
| **Manifest** | DNA profile | Prompt config | Provider-specific generation |
| **Deploy** | Manifest | Running clone | Runtime execution |

## Metrics & Scores

### Fidelity Score (0-100)

Measures DNA profile completeness and quality:
- **90-100**: High fidelity, comprehensive coverage
- **70-89**: Good fidelity, minor gaps
- **50-69**: Moderate fidelity, significant gaps
- **0-49**: Low fidelity, insufficient data

### Confidence Score (0.0-1.0)

Per-trait confidence based on:
- Sample size (amount of source content)
- Consistency (agreement across sources)
- Quality (source reliability)

### Similarity Score (0.0-1.0)

Measures clone response alignment with original content:
- Semantic similarity (embedding distance)
- Stylistic similarity (pattern matching)
- Content accuracy (fact verification)

### Temperature

LLM sampling parameter affecting creativity:
- **0.0-0.3**: Focused, deterministic (factual responses)
- **0.4-0.7**: Balanced (conversational)
- **0.8-1.0**: Creative, varied (brainstorming)
