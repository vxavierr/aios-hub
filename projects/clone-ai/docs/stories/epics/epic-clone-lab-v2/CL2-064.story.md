# Clone Lab v2.0 - Self-Evolving AI Personality Cloning

## Overview

Clone Lab v2.0 transforms static AI personality cloning into a **self-evolving platform** with:

- **8 Analytical Minds** - Specialized analysis by personality dimension
- **63 Validation Tasks** - Granular quality validation
- **6 Human Checkpoints** - Human-in-the-loop control
- **Meta-Cognition Layer** - Self-assessment and auto-improvement
- **Process Evolution Tracking** - Full audit trail of changes

## Quick Start

\`\`\`bash
# Install
npm install -g @clone-lab/cli

# Initialize
clone-lab init

# Extract data
clone-lab extract ./sources --output ./data

# Analyze (runs 8 Minds)
clone-lab analyze ./data --output ./analysis

# Create clone DNA
clone-lab synthesize ./analysis --name "MyClone" --output ./clone.dna.json

# Generate manifest
clone-lab manifest ./clone.dna.json --provider claude

# Run clone
clone-lab run ./manifests/my-clone --provider claude
\`\`\`
