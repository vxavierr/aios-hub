## CLI Commands Preview

```bash
# Initialization
clone-lab init

# Extraction
clone-lab extract ./chat-logs --format json
clone-lab extract https://youtube.com/@channel --type youtube

# Analysis
clone-lab analyze ./extracted-data.json --output ./analysis.json

# DNA Synthesis
clone-lab synthesize ./analysis.json --name "MyClone" --output ./my-clone.dna.json

# Manifest Generation
clone-lab manifest ./my-clone.dna.json --provider claude --output-dir ./manifests

# Runtime
clone-lab run ./manifests/my-clone --provider claude
```
